# Pokédex Web App

A simple and interactive Pokédex webpage built using HTML, CSS, and JavaScript**.  
It fetches Pokémon data from the [PokéAPI](https://pokeapi.co/) and displays it in a clean, user-friendly interface.  
Users can search for any Pokémon by name to view basic information such as its image, type, height, and weight.

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- PokéAPI ([https://pokeapi.co/](https://pokeapi.co/))
